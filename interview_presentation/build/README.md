# Build / Export Notes

No slide build tooling is configured in this repository.

## Option 1: Marp CLI (recommended)

If you have Marp installed locally:

```bash
npx @marp-team/marp-cli ../deck.md --allow-local-files --html -o deck.html
npx @marp-team/marp-cli ../deck.md --allow-local-files --pdf -o deck.pdf
```

Run from:

```bash
cd interview_presentation/build
```

## Option 2: VS Code Markdown Preview

1. Open `interview_presentation/deck.md` in VS Code.
2. Use Markdown preview.
3. Print to PDF from preview if needed.

## Inputs used for visuals

Visuals are generated by:

```bash
python interview_presentation/make_plots.py
```

This writes PNG/SVG files to `interview_presentation/assets/`.
